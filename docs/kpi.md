# KPIs — A.R.I.

## Cost
| Metric | Purpose |
|---------|----------|
| Total Tickers Tracked | Control API cost growth |
| Total Emails Sent | Volume / engagement indicator |
| LLM Tokens Used (per vendor) | Track usage vs free limits |
| Vendor Calls per Day | Detect over-use of NewsAPI / BSE / NSE |

## User Experience
| Metric | Description |
|---------|--------------|
| Email Open Rate | Opens / Sent |
| Avg Ratings Trend (D/W/M) | Rolling satisfaction |
| Overall Avg Rating | Combined score |
| Feedback Volume | Count (24 h / 7 d / 30 d / All) |

## Reliability & Operations
| Metric | Description |
|---------|--------------|
| Delivery Success % | Emails sent / attempted |
| Generation Latency (ms) | Average and trend |
| API Error Rate | Failures / Calls (after retries) |
| Ingestion → Delivery Time (min) | End-to-end efficiency |
| Missed Emails | Alerts on send failures |
